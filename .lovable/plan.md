

# TrendZone — Fashion E-Commerce Landing Page

## Overview
A bold, fashion-forward landing page for **TrendZone** — a modern fashion brand. Frontend-only with placeholder images and content, following the vibrant color-blocking design system and reference images provided.

---

## 1. Navigation Bar
- Fixed top nav with glassmorphism effect (frosted glass background)
- **TrendZone** logo on the left
- Centered nav links: Home, New Arrival, Shop, Contact, About Us
- Right side: Search icon, Cart icon, "Sign In" outlined button
- Hamburger menu on mobile

## 2. Hero Section
- Large bold headline: **"Elevate Your Style With Bold Fashion"**
- Asymmetric bento-box grid of fashion images with vibrant color-blocked backgrounds (orange, green, yellow, sky blue, mint)
- Rounded image cards at varying sizes, creating dynamic visual rhythm
- Rotating "Learn about us" circular text with play icon
- Avatar group with "+N" counter
- "Explore Collections" CTA button (dark, pill-shaped)

## 3. Testimonial + Lifestyle Section
- Split layout below hero
- Left: Quote with large quotation mark, review text, and handwritten-style signature
- Right: Numbered section header ("01 — Lifestyle") with arrow, bold subtitle "Set Up Your Fashion With The Latest Trends"

## 4. Categories Section
- Full-width list of categories (Clothes, Shoes, Bags, Accessories) with divider lines
- Arrow on each row that animates on hover
- Product image preview on the right side

## 5. Ready to Wear / Product Grid
- Section header with "SHOP NOW →" link
- 3-column product card grid with rounded image containers
- Product name and price below each card
- Hover: subtle image scale effect

## 6. Looks Gallery
- Asymmetric masonry-style image grid
- Description text alongside images
- "Full Gallery →" link

## 7. New Styles Section
- Bold text overlay card ("New Styles For A Modern Look")
- Small product category cards (Jacket, Back Pleat, Tote Bag) with labels

## 8. Happy Voices / Testimonials
- Customer review cards with star ratings
- Profile avatars and names
- Navigation arrows for carousel

## 9. Instagram Feed
- "@TrendZone" social media section
- Row of fashion images in a horizontal scroll or grid

## 10. Footer
- Multi-column layout with brand info, navigation links, contact details
- Social media icons
- Newsletter signup input

---

## Design Details
- **Fonts**: Outfit (headings), Inter (body), Playfair Display (accent/signatures)
- **Colors**: Cream background (#F5F3ED), vibrant accent blocks (orange, green, yellow, sky blue), dark text (#1A1A1A)
- **Border Radius**: 24–40px on cards, 50px on buttons
- **Animations**: Scroll-reveal fade-up on sections, hover scale on images, smooth button hover lifts
- **Responsive**: Mobile-first with stacked layouts, hamburger nav, fluid typography using clamp()
